API Docs

Apps — CRUD

GET /api/apps List all apps

Returns array of all registered apps.

Try it
GET /api/apps/:id Get one app

Returns a single app by ID.

Try it
POST /api/apps Register new app

Register a new app. Auto-creates: port assignment, Caddy proxy, LaunchAgent, health check.

Body
FieldTypeDescription
idstringrequiredUnique app identifier (kebab-case)
namestringoptionalDisplay name (defaults to id)
localPathstringoptionalAbsolute path to project root
localUrlstringoptionalAuto-assigned if omitted
healthUrlstringoptionalDefaults to localUrl
repostringoptionalGitHub repo URL
startCommandstringoptionalDefaults to "npm run dev"
iconstringoptionalURL/path to app icon
logPathstringoptionalDefaults to /tmp/{id}.log
Example
curl -X POST http://localhost:9876/api/apps \
  -H "Content-Type: application/json" \
  -d '{"id":"my-app","localPath":"/Users/me/Sites/my-app"}'Copy
PUT /api/apps/:id Update app

Update any fields on an existing app. Only provided fields are changed.

Example
curl -X PUT http://localhost:9876/api/apps/my-app \
  -H "Content-Type: application/json" \
  -d '{"localUrl":"http://localhost:4000","icon":"/favicons/my-app.svg"}'Copy
DELETE /api/apps/:id Remove app + teardown infra

Deletes app from DB, removes Caddy entry, and unloads LaunchAgent.

Example
curl -X DELETE http://localhost:9876/api/apps/my-appCopy

Status & Monitoring

GET /api/status Dashboard status (all apps + machine info)

Returns all apps with live status, LAN URLs, machine hostname, model, and IP.

GET /api/events SSE stream (live updates)

Server-Sent Events stream. Events: update (status change), reload (config change), screenshots_done.

GET /api/log/:id Last 30 lines of app log

Returns { lines: [...] } from the app's log file.

POST /api/start/:id Start app via LaunchAgent

Loads and starts the app's LaunchAgent plist.

Machine Sync

GET /api/machine Machine identity

Returns hostname, model, LAN IP, port, and app count.

GET /api/apps/export Portable app list for syncing

Returns app list without machine-specific paths. Used by /api/sync.

POST /api/sync Pull apps from another machine

Connects to a remote Local Apps instance and merges its app list. Adds missing apps, syncs ports/repos/icons.

Body
FieldTypeDescription
remotestringrequiredIP or IP:port of remote machine (e.g. "10.0.0.218")
Example
curl -X POST http://localhost:9876/api/sync \
  -H "Content-Type: application/json" \
  -d '{"remote":"10.0.0.218"}'Copy

Screenshots

GET /api/screenshots/:id Get screenshots for an app

Returns screenshot list and capture timestamp.

POST /api/screenshots/:id Capture screenshots for an app

Triggers the screenshot bot for the specified app. Runs in background.

Example
curl -X POST http://localhost:9876/api/screenshots/bhengCopy
GET /api/screenshots-status Running screenshot jobs

Returns currently running screenshot capture jobs.

DELETE /api/screenshot Delete a screenshot
Body
FieldTypeDescription
